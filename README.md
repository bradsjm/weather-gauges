The steelseries library is a collection of animated gauges components related to navigation.

## History and Credits

Originally SteelSeries [Java Swing](https://harmoniccode.blogspot.com/2010/08/java-swing-component-library.html) version created by HanSolo in 2010 then updated to the javascript canvas based [SteelSeries-Canvas](https://github.com/HanSolo/SteelSeries-Canvas) from his original in 2013. A web component steelseries implemented was created by [Nicolas Vanhoren](https://github.com/nicolas-van/steelseries) in 2019.
